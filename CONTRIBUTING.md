# Contributing to Smart Nutrition Tracker

Thank you for your interest in contributing! 🎉

## How to Contribute

### 1. Fork the Repository
Click the **Fork** button at the top right of this page.

### 2. Clone Your Fork
git clone https://github.com/your-username/updated-smart-nutrition-tracker.git
cd updated-smart-nutrition-tracker

### 3. Create a New Branch
git checkout -b feature/your-feature-name

### 4. Make Your Changes
- Write clean, readable code
- Follow existing code style
- Test your changes before committing

### 5. Commit Your Changes
git commit -m "feat: describe your change here"

### 6. Push to Your Fork
git push origin feature/your-feature-name

### 7. Open a Pull Request
Go to the original repository and click **"New Pull Request"**.

## Reporting Bugs
- Open an issue on GitHub
- Describe the bug clearly
- Include steps to reproduce it

## Commit Message Format
Use this format for commit messages:

- feat: adding a new feature
- fix: fixing a bug
- docs: documentation changes
- chore: maintenance tasks
- test: adding tests

## Code of Conduct
Please be respectful and inclusive in all interactions.
Be kind, collaborative and constructive in all feedback.
